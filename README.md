# Java Variables Program

## Overview
A simple Java program that takes two integer inputs and performs basic arithmetic operations.

## Usage
1. Compile the program:
   ```sh
   javac Variables.java
   ```
2. Run the program:
   ```sh
   java Variables
   ```
3. Enter two integers when prompted.
